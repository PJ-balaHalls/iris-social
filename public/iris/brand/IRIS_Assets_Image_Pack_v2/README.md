# IRIS — Pack de Assets de Imagem (v2)

Este pacote foi refeito para atender ao pedido de **assets visuais reais**, com aparência de imagem premium, e **não** componentes simples ou flores "desenhadas como código".

## O que tem neste pack

### Categorias
- **backgrounds/**
- **imagens-oficiais/**
- **ilustracoes/**
- **estados/**
- **cards/**
- **banners/**

### Breakpoints
Cada categoria está organizada em:
- **mobile/**
- **tablet/**
- **desktop/**

## Formatos entregues
Em cada pasta, cada asset possui:
- **.jpg** → imagem em alta qualidade
- **.svg** → versão em contêiner SVG com a imagem incorporada em alta resolução

> Observação importante: como o pedido foi por “imagem JPG convertida em SVG”, os arquivos SVG deste pack usam a imagem incorporada dentro do SVG. Isso preserva compatibilidade de layout e uso vetorial de contêiner, sem parecer um elemento HTML simples.

## Direção visual usada
Baseado no **IRIS Design System**:
- paleta suave com off-white, verde floresta, sálvia e lilás
- estética íntima, editorial, elegante e calma
- fotografia natural e ilustração premium
- uso pensado para mobile-first, tablet e desktop
